# Cuantificación de Daños Antitrust

**Microeconometría Aplicada a la Cuantificación de Daños por Competencia Desleal y Prácticas Anticompetitivas**

*Carlos de Anta Puig*

Manual profesional de métodos econométricos para la cuantificación de daños económicos en procedimientos judiciales y arbitrales por infracciones del Derecho de la competencia y competencia desleal.

## Contenido

- **Parte I** — Fundamentos (Caps. 1–4): daño económico, conductas ilícitas, causalidad, datos
- **Parte II** — Métodos de estimación (Caps. 5–10): antes-y-después, comparables, DID, sobreprecio, passing-on, lucro cesante
- **Parte III** — Métodos avanzados (Caps. 11–15): análisis financiero, variables instrumentales, panel, control sintético, regresión cuantílica
- **Parte IV** — Del modelo al informe pericial (Caps. 16–19): actualización financiera, sensibilidad, errores frecuentes, redacción y defensa
