15  Regresión cuantílica y efectos heterogéneos

15.1 El daño no afecta igual a todos

Los métodos vistos hasta ahora estiman un efecto medio de la conducta sobre los precios, las ventas o los márgenes. Pero en muchos casos antitrust, el daño no es homogéneo: algunos compradores pagan un sobreprecio mayor que otros, algunas empresas sufren más la exclusión que otras, algunos mercados se ven más afectados que otros. La regresión cuantílica permite examinar cómo varía el efecto de la conducta a lo largo de la distribución de la variable de resultado.

En acciones colectivas, esta heterogeneidad es especialmente relevante: el tribunal puede necesitar saber no solo cuánto daño sufrió el comprador medio, sino cuánto sufrió el comprador del percentil 10, del 50 o del 90.

15.2 La intuición: mirar más allá del promedio

Code 
set.seed(1515)

n <- 400
tamano <- runif(n, 10, 500)   # tamaño de la transacción
cartel <- c(rep(0, 200), rep(1, 200))

# Efecto heterogéneo: compradores pequeños sufren más
efecto_cartel <- cartel * (18 - 0.02 * tamano + rnorm(n, 0, 2))
precio <- 50 + 0.05 * tamano + efecto_cartel + rnorm(n, 0, 3)

df_qr <- tibble(
  obs = 1:n,
  precio = round(precio, 2),
  tamano = round(tamano, 2),
  cartel = cartel
)

ggplot(df_qr, aes(x = tamano, y = precio, color = factor(cartel))) +
  geom_point(alpha = 0.5, size = 1.8) +
  geom_smooth(method = "lm", se = FALSE, linewidth = 0.8) +
  scale_color_manual(values = c("0" = "#3498db", "1" = "#e74c3c"),
                     labels = c("Sin cártel", "Con cártel")) +
  labs(x = "Tamaño de transacción", y = "Precio", color = NULL) +
  theme_minimal(base_size = 13) +
  theme(legend.position = "bottom", panel.grid.minor = element_blank())

15.3 La idea econométrica

La regresión cuantílica estima el efecto de los regresores sobre un cuantil concreto de la distribución condicional de \(Y\):

\[ Q_\tau(Y_i | X_i) = \alpha_\tau + \beta_\tau \, \text{Cartel}_i + \gamma_\tau X_i \]

donde \(\tau \in (0, 1)\) indica el cuantil. Para \(\tau = 0.5\) se obtiene la mediana; para \(\tau = 0.10\) y \(\tau = 0.90\), los extremos de la distribución.

Code 
taus <- c(0.10, 0.25, 0.50, 0.75, 0.90)

resultados_qr <- map_dfr(taus, function(tau) {
  fit <- rq(precio ~ cartel + tamano, data = df_qr, tau = tau)
  tibble(
    Cuantil = tau,
    `Sobreprecio` = round(coef(fit)["cartel"], 2),
    `Coef. tamaño` = round(coef(fit)["tamano"], 4)
  )
})

resultados_qr |>
  knitr::kable(format = "pipe", align = c("r", "r", "r"))

La Table 15.1 muestra que el sobreprecio no es uniforme: varía a lo largo de la distribución de precios. Esta evidencia es fundamental para una acción colectiva donde distintos compradores pueden haber sufrido daños de distinta magnitud.

15.4 Visualización del efecto por cuantiles

Code 
taus_fine <- seq(0.05, 0.95, by = 0.05)
coefs_qr <- map_dfr(taus_fine, function(tau) {
  fit <- rq(precio ~ cartel + tamano, data = df_qr, tau = tau)
  se <- tryCatch(summary(fit, se = "nid")$coefficients["cartel", "Std. Error"],
                 error = function(e) NA_real_)
  tibble(tau = tau, coef = coef(fit)["cartel"], se = se)
})

ols_coef <- coef(lm(precio ~ cartel + tamano, data = df_qr))["cartel"]

ggplot(coefs_qr, aes(x = tau, y = coef)) +
  geom_ribbon(aes(ymin = coef - 1.96 * se, ymax = coef + 1.96 * se),
              fill = "#3498db", alpha = 0.2) +
  geom_line(color = "#2c3e50", linewidth = 1) +
  geom_point(color = "#2c3e50", size = 2) +
  geom_hline(yintercept = ols_coef, color = "#e74c3c", linetype = "dashed", linewidth = 0.8) +
  annotate("text", x = 0.85, y = ols_coef + 0.8, label = "OLS (media)",
           color = "#e74c3c", size = 3.5) +
  labs(x = "Cuantil", y = "Sobreprecio estimado") +
  theme_minimal(base_size = 13) +
  theme(panel.grid.minor = element_blank())

La Figure 15.2 muestra cómo el sobreprecio varía con el cuantil. La línea roja horizontal representa el efecto medio (OLS). Cuando la curva de cuantiles se desvía de esa línea, hay heterogeneidad en el daño que el efecto medio no captura.

15.5 Implementación en R

Code 
library(quantreg)

# Regresión cuantílica para varios cuantiles
taus <- c(0.10, 0.25, 0.50, 0.75, 0.90)
fits <- lapply(taus, function(tau) rq(precio ~ cartel + tamano, data = datos, tau = tau))

# Extraer coeficientes
sapply(fits, function(f) coef(f)["cartel"])

# Gráfico de proceso cuantílico
plot(summary(rq(precio ~ cartel + tamano, data = datos, tau = seq(0.05, 0.95, 0.05))))

15.6 Relevancia jurídica

La regresión cuantílica tiene una aplicación directa en acciones colectivas de daños antitrust, donde el tribunal debe determinar no solo el daño medio sino la distribución del perjuicio entre los miembros de la clase. Si el sobreprecio es heterogéneo —por ejemplo, mayor para compradores pequeños que para grandes—, la regresión cuantílica proporciona la evidencia necesaria para una cuantificación individualizada.

15.7 Sensibilidad

Code 
sens_qr <- tibble(
  Método = c("OLS (media)", "Mediana (q50)", "q10 (compradores baratos)", "q90 (compradores caros)"),
  Sobreprecio = c(
    round(coef(lm(precio ~ cartel + tamano, data = df_qr))["cartel"], 2),
    round(coef(rq(precio ~ cartel + tamano, data = df_qr, tau = 0.5))["cartel"], 2),
    round(coef(rq(precio ~ cartel + tamano, data = df_qr, tau = 0.1))["cartel"], 2),
    round(coef(rq(precio ~ cartel + tamano, data = df_qr, tau = 0.9))["cartel"], 2)
  )
)

sens_qr |>

15.8 Errores frecuentes

El error más común es interpretar la regresión cuantílica como si estimara el efecto sobre diferentes subgrupos de compradores. No es así: estima el efecto sobre diferentes puntos de la distribución condicional del precio. El segundo error es ignorar que los errores estándar de la regresión cuantílica requieren métodos específicos (bootstrap o kernel) y que los intervalos de confianza asintóticos pueden ser poco fiables en muestras pequeñas.

15.9 Lo que un tribunal debe retener

La regresión cuantílica revela que el daño no es necesariamente igual para todos. En acciones colectivas, esta información es esencial para cuantificar el perjuicio de forma equitativa. Cuando el perito muestra que el sobreprecio varía según el tamaño de la transacción, la localización del comprador o la calidad del producto, el tribunal dispone de una base para una indemnización diferenciada.
