18  Errores frecuentes en informes periciales de daños

18.1 Aprender del error ajeno

He revisado, a lo largo de mi carrera, centenares de informes periciales de daños —propios y ajenos, de demandantes y demandados, en cárteles, abusos de dominio y competencia desleal. Los errores que recojo en este capítulo no son hipotéticos: son errores que he visto, que en algunos casos he cometido y que, en todos los casos, tienen consecuencias cuantificables sobre el resultado.

Este capítulo recorre los diez errores más frecuentes en la práctica pericial de daños antitrust. Para cada uno, se explica el mecanismo del error, se ilustra con un ejemplo numérico que muestra cómo cambia el resultado y se ofrece la corrección correspondiente.

18.2 Error 1: Confundir sobreprecio con margen

Code 
tibble(
  Concepto = c("Precio observado", "Coste", "Margen empresarial", "Precio competitivo (estimado)", "Sobreprecio real"),
  Valor = c("100", "70", "30 (30%)", "85", "15 (17.6%)")
) |>
  knitr::kable(format = "pipe", align = c("l", "r"))

Un perito que utilice el margen del 30% como proxy del sobreprecio duplicará la cuantificación real del daño.

18.3 Error 2: Período previo contaminado

Si el cártel comenzó antes de la fecha oficialmente identificada, el período “previo” ya contiene el sobreprecio. El contrafactual queda sesgado al alza y el daño se subestima.

Code 
tibble(
  Escenario = c("Período limpio (correcto)", "Período contaminado (erróneo)"),
  `Daño estimado` = c(500, 350),
  `Sesgo` = c("—", "-30%")
) |>
  knitr::kable(format = "pipe", align = c("l", "r", "r"))

18.4 Error 3: Ignorar la estacionalidad

Comparar enero (mes bajo) con julio (mes alto) sin controlar estacionalidad produce un contrafactual artificialmente inflado o deflactado.

18.5 Error 4: Comparable no realmente comparable

Un mercado con regulación distinta, estructura de costes diferente o composición de clientes heterogénea no es un yardstick válido, aunque venda el mismo producto.

18.6 Error 5: Violación de tendencias paralelas no detectada

Aplicar DID cuando las trayectorias ya divergían antes del tratamiento invalida la estimación causal. La verificación gráfica y los placebos pretratamiento son imprescindibles.

18.7 Error 6: Endogeneidad ignorada

Estimar por OLS cuando precio y cantidad se determinan simultáneamente sesga el sobreprecio. La diferencia entre OLS e IV puede ser sustancial (como se mostró en el Capítulo 12).

18.8 Error 7: Omisión del efecto volumen

Code 
tibble(
  Componente = c("Sobreprecio (A)", "Efecto volumen (B)", "Daño total (A+B)"),
  Valor = c(1296, 252, 1548),
  `% del total` = c("83.7%", "16.3%", "100%")
) |>
  knitr::kable(format = "pipe", align = c("l", "r", "r"))

Omitir el efecto volumen deja fuera el 16% del daño en este ejemplo.

18.9 Error 8: Tasa de descuento inadecuada

Code 
tibble(
  Tasa = c("0% (sin actualizar)", "2%", "4%", "8%"),
  `Daño actualizado` = c(1800, 1863, 1919, 2034),
  `Diferencia vs. 4%` = c("-6.2%", "-2.9%", "—", "+6.0%")
) |>
  knitr::kable(format = "pipe", align = c("l", "r", "r"))

18.10 Error 9: Ausencia de análisis de sensibilidad

Presentar un único número sin mostrar cómo varía con los supuestos es la forma más rápida de perder credibilidad ante un tribunal competente.

18.11 Error 10: Cherry picking

Seleccionar el período, la especificación o el grupo de control que produce el resultado más favorable al cliente es una mala práctica que la contraparte detectará. El perito debe justificar sus elecciones con criterios económicos y mostrar las alternativas.

18.12 Resumen

Code 
tibble(
  `Nº` = 1:10,
  Error = c("Confundir sobreprecio con margen",
            "Período previo contaminado",
            "Ignorar estacionalidad",
            "Comparable inadecuado",
            "Tendencias paralelas no verificadas",
            "Endogeneidad ignorada",
            "Omisión del efecto volumen",
            "Tasa de descuento inadecuada",
            "Sin análisis de sensibilidad",
            "Cherry picking"),
  Consecuencia = c("Sobreestimación del daño",
                    "Subestimación del daño",
                    "Sesgo en cualquier dirección",
                    "Contrafactual inválido",
                    "Estimación causal no creíble",
                    "Sesgo en el sobreprecio",
                    "Subestimación ~15-20%",
                    "Sesgo en la actualización",
                    "Pérdida de credibilidad",
                    "Vulnerabilidad a impugnación")
) |>
  knitr::kable(format = "pipe", align = c("r", "l", "l"))

18.13 Lo que un tribunal debe retener

Este capítulo es, en esencia, una lista de verificación. Un tribunal que conozca estos diez errores estará en condiciones de evaluar la solidez de cualquier informe pericial de daños. Un perito que los evite estará construyendo una cuantificación más difícil de impugnar.
